# Void Apple Mod

一个为 Minecraft 1.20.1 设计的 Forge 模组，添加了**虚空苹果**。

## 功能

- 🍎 **虚空苹果**：吃掉后会传送到末地黑曜石平台
- 🎨 **自定义模型与贴图**：独特的紫色苹果外观
- 📦 **创造模式标签页**：方便获取物品
- 🦑 **潜影贝掉落**：击杀潜影贝有 5% 概率掉落虚空苹果

## 获取方式

- 创造模式物品栏 → "Void Apple" 标签页
- 命令：`/give @p voidapple:void_apple`
- 击杀潜影贝（5% 概率掉落）

## 合成配方

- 1 个苹果 + 8 个紫颂果 = 1 个虚空苹果

## 模组信息

- Minecraft 版本：1.20.1
- Forge 版本：47.2.0
- 模组 ID：`voidapple`
